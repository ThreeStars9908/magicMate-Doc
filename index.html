Introduction

Thank you for your interest in MagicMate.

Event List

Event List functionality is a feature typically found in various applications, such as calendar apps, task management tools, or event planning platforms. It allows users to create, manage, and track events or tasks within a specific timeframe. The Event List provides an organized view of all the scheduled events, making it easier for users to stay informed and keep track of their commitments.

  • Creating Events:

    Users can create new events by providing essential details such as the event title, date, start time, end time, location (if applicable), and any additional notes or descriptions. This information is usually entered through a user-friendly interface or by using natural language processing to interpret text-based inputs.

  • Managing Events:

    Once an event is created, it can be managed within the Event List. Users can easily view, edit, or delete events as needed. Editing an event allows users to modify details like the event title, date, time, or location, while deleting an event removes it from the list entirely.

  • Date and Time Display:

    The Event List typically displays events chronologically, based on their date and time. It may use different visual indicators to differentiate upcoming, ongoing, or completed events. This helps users quickly identify which events are happening soon or require immediate attention.

  • Sharing and Collaboration:

    In some cases, Event Lists provide sharing and collaboration features. Multiple users can access and contribute to the same event list, making it ideal for team-based project management or event planning. Users can assign tasks, set deadlines, and receive real-time updates on progress.

  • Search Functionality:

    Large Event Lists can benefit from a search function that allows users to quickly find specific events based on keywords or criteria. This is particularly useful when dealing with extensive event histories or when searching for past events.

Mandatory setup

Change App Logo

You can generate app icon from this website https://app icon.co.

  • Go to <project>/assets/ and replace logo1.png with your own logo.
  • Then go to /android/app/src/main/res and replace all mipmap folder with your <generated icon>/android folder.
  • Again go to /ios/Runner and replace Assets.xcassets with your generated Assets.xcassets folder.

Change App Name

  1. Change the value of label from <project>/android/app/src/main/AndroidManifest.xml

     android:label="My App"

  2. Change the value of CFBundleName from <project>/iOS/Runner/info.plist

    <key>CFBundleName</key>
 <string>My App</string>

Change Base URL

Must remember that don’t put slash(/) at the end of your base url. Use your admin url as base url. First you have to install your admin panel. For example: If your admin url is https://Subdomain.your_domain.com/ then base url will be https://Subdomain.your_domain.com. Open /lib/Api/config.dart and replace baseurl variable value with your own URL.

 static const String baseurl = 'https://Subdomain.your_domain.com/';

Change App Package

First you have to find out the existing package name. You can find it out from top of /app/src/main/AndroidManifest.xml file. Now right click on project folder from android studio and click on replace in path. You will get a popup window with two input box. In first box you have to put existing package name that you saw in AndroidManifest.xml file previously and write down your preferred package name in second box and then click on Replace All button.
